<h5>Задача</h5>

Реализовать приложение «Телефонный справочник», в котором можно искать, добавлять и удалять абонентов, а также просматривать весь список абонентов.

Для хранения данных использовать любую БД, структура данных следующая:

<ol>
    <li>ФИО абонента</li>
    <li>Телефон</li>
    <li>Адрес</li>
</ol>

Поиск осуществляется по любому из полей. 

Все операции реализовать в виде RESTful веб-сервисов, например:

<ol>
    <li>Добавление абонента: POST запрос на адрес: localhost:8080/telephone-dir/add с данными абонента.</li>
    <li>Удаление абонента с идентификатором 1: POST запрос на адрес: localhost:8080/telephone-dir/delete/1.</li>
</ol>

Все ответы сервера реализовать в виде JSON.

Приложение должно быть упаковано в war файл и запускаться в Tomcat 8 либо использовать Spring Boot.

Для всех публичных методов необходимо написать юнит-тесты, также необходимо кратко задокументировать, что этот код делает (в виде java-doc'ов)

Приложение реализовать с использованием следующих технологий:

<ol>
    <li>Серверная часть: Java 8 и фреймворк Spring.</li>
    <li>Клиентская часть: Thymeleaf, HTML, CSS, jQuery // Или на свой вкус и возможности</li>
    <li>Сборка приложения: Maven или Gradle.</li>
</ol>

Результат прислать zip архивом либо ссылку на открытый репозитарий.
